1. Загрузить с все содержимое папки "скрипт" на сервер.
2. Загрузить базу данных из папки "база" на ваш mysql сервер.
3. Открыть на сервере файл setting.php (он в корне), и за вписать данные к доступу к базе данных.
4. Теперь  можете настроить сайт, для этого перейдите по адресу http://vashsite.ru/admin доступ к админке логин: admin пароль: admin
5. Для изменения пароля к админке надо перейти в базу на сервере и открыть таблицу settings и найти сторку admin_pass и сменить значение admin на любое другое.
6. Все ключи и id API PAYEER вы сможете найти на их сайте.
